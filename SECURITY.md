## Reporting a Vulnerability

If you discover a security vulnerability in Vulnora itself (e.g., in the backend, authentication, or scanning engine), please report it responsibly.

**Do not** create a public GitHub issue.

Instead, send a detailed report to:

**Email**: [secguy22@gmail.com] (replace with your real contact email)

or open a **private vulnerability report** on GitHub (if enabled).

### What to Include
- Description of the vulnerability
- Steps to reproduce
- Potential impact
- Suggested fix (if known)

We will acknowledge your report within 48 hours and aim to resolve critical issues quickly.

Thank you for helping keep Vulnora secure.
